# FBT
Python script to obtain an EAAAA token for Facebook.

## Installation
```
git clone https://github.com/kiryudev/FBT
cd FBT
python FBT.py
```
## How to use
* Execute the script.
* Input your email or username and password.

## Dependencies
- requests
- random
- string
- uuid
- rich
- sys
- os
